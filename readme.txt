El proyecto esta basado en una Hamburgueseria artesanal, cuenta con home, menú, contacto, registro, login, about us, experiencias, y Gestor de usuarios (CRUD implementado).
En este repositorio encontrara el proyecto backend y frontend. Además de la docuemtación donde esta el taller frontend y backend desarrollados en clase.
